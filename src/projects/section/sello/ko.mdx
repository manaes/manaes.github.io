### 셀로 서비스

- LGU+ 보상판매를 위한 서비스 앱 개발
- 단말기 상태 체크 및 키오스크와 통신을 담당하는 셀로링크 앱
- 시세 및 기타 서비스 제공을 위한 셀로 앱

#### 셀로 링크
 
- 기기의 정상여부를 판단하기 위해, 시스템 정보 + 여러 시스템 센서 체크 로직 연동
- 키오스크와 통신을 위한 API Polling + UI 연동
- 기기 일부 설정값 로딩을 위한 이미지 OCR 기능 구현
- MDM 기능 연동 (기기초기화 / IMEI / 기능 제한 등)

#### 셀로
 
- 중고휴대폰 거래를 위한 메인 앱 (웹앱)
- UI 및 인증 / 카메라 연동을 위한 Webkit Message Handler 구현
- 소셜 로그인 기능 구현
- 기기 상태 촬영을 위한 촬영가이드 및 카메라 기능 구현

### Tech Stack (iOS)

- Swift, Storybard, Snapkit, Firebase, MDM
