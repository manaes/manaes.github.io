### Role

- 코박팀 리더 / iOS 개발자

### 주요 기능
 
- 블럭체인 커뮤니티 / 뉴스 + 개인 블럭체인 지갑 연동 서비스
- v2.0 개발 및 유지보수
- 2000개+ 토큰 시세 및 정보 노출
- 블럭체인 지갑 송금 및 스마트컨트렉 연동
- 앱내 디자인 시스템 구축 및 연동
- 자동화 테스트 및 배포 구축

### Tech Stack (iOS)

- RxSwift, Storyboard, Snapkit, Moya, Quick/Nimble, Web3Swift, WalletConnect

### v2.0 개발 및 유지보수

- 기존 v1.0의 경우, 여러 레거시 코드가 혼재해있었음 (objective-c / swift / swiftUI)
- 단일 구조로 변경하기 위해, RxSwift + MVVM 구조로 변경
- Moya를 사용한 API Provider 패턴 사용
- 대부분 Storyboard를 사용했으나, 필요에 따라 코드/xib/swiftUI를 사용함
- 오픈소스는 SPM을 주로 사용, 일부 carthage + rome을 통한 빌드 속도 개선

### 2000개+ 토큰 시세 및 정보 노출

- 일부 토큰의 경우, websocket을 통한 실시간 시세 제공
- 서버에서 주기적으로 수집한 시세 CSV 파일을 제공 -> RealmDB를 통해 서비스

### 블럭체인 지갑 송금 및 스마트컨트렉 연동

- 키스토어/니모닉/PK를 사용한 개인지갑 import
- 이더리움/메틱 주요 토큰에 대해, balance/send 등의 기능 구현
- 자체 토큰(CBK)의 smart contract 연동 (swap/ido/governance)
- [web3swift](https://github.com/web3swift-team/web3swift), [walletConnect](https://walletconnect.network/)를 사용해 구현

### 앱내 디자인 시스템 구축 및 연동

- 기존 디자인 관련 Document가 존재하지 않아, 처음부터 디자인 시스템을 재구축함
- 색상/폰트/컴포넌트 디자인 구현 및 모듈식 코드 구현
- DarkMode 및 Dynamic Type 구현

### 자동화 테스트 및 배포 구축

- mock, quick/nimble을 주로 사용
- Moya - sample data를 사용한 네트워크 API 검증
- Unit/UI 테스트 + Fastlane을 통한 자동화 테스트 (code coverage 80% 이상)
- 4시간주기 CI/CD 내부 배포 및 슬랙 결과 연동
